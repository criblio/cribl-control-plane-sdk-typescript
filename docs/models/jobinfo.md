# JobInfo

Detailed information about a job, including its configuration, status, and statistics.

## Example Usage

```typescript
import { JobInfo } from "cribl-control-plane/models";

let value: JobInfo = {
  args: {
    executor: {
      type: "<value>",
    },
    run: {},
  },
  id: "<id>",
  stats: {},
  status: {
    state: 7,
  },
};
```
