# PipelineConf

Configuration for the Pipeline, including functions and settings.

## Example Usage

```typescript
import { PipelineConf } from "cribl-control-plane/models";

let value: PipelineConf = {
  functions: [
    {
      id: "notifications",
      conf: {
        id: "<id>",
        field: "<value>",
        deduplicate: false,
      },
    },
  ],
};
```
