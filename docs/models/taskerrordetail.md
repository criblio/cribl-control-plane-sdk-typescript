# TaskErrorDetail

Task error details. May include a nested <code>reason</code> for wrapped errors and additional properties from the original error.

## Example Usage

```typescript
import { TaskErrorDetail } from "cribl-control-plane/models";

let value: TaskErrorDetail = {
  message: "<value>",
};
```
