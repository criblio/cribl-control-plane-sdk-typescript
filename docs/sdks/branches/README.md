# Branches
(*versions.branches*)

## Overview

### Available Operations

* [list](#list) - List all branches in the Git repository used for Cribl configuration
* [get](#get) - Retrieve the name of the Git branch that the Cribl configuration is checked out to

## list

get the list of branches

### Example Usage

<!-- UsageSnippet language="typescript" operationID="getVersionBranch" method="get" path="/version/branch" -->
```typescript
import { CriblControlPlane } from "cribl-control-plane";

const criblControlPlane = new CriblControlPlane({
  serverURL: "https://api.example.com",
  security: {
    bearerAuth: process.env["CRIBLCONTROLPLANE_BEARER_AUTH"] ?? "",
  },
});

async function run() {
  const result = await criblControlPlane.versions.branches.list();

  console.log(result);
}

run();
```
